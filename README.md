## 🏁 Getting Started <a name = "getting_started"></a>

```
git clone git@github.com:juanmorenomotta/deno-restfulapi-en-memoria.git
```

### Prerequisites

Se requiere Deno 1.4.2.

```
deno install -qAf --unstable https://deno.land/x/denon@2.3.3/denon.ts 

denon start
```
